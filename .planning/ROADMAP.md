# Roadmap: OpenAI Responses Compatibility Proxy

## Overview

Deliver a production-grade compatibility proxy that preserves Anthropic Messages semantics on top of OpenAI Responses. The v1.0 roadmap focused on core request/response parity, token counting alignment, privacy-first observability, and streaming/tool-use parity so Claude Code can run without semantic breaks.
